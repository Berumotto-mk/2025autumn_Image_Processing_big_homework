CycleGAN 双域 · 中文工作流

墨境实验室，托起风景与人像的双重想象

使用已训练的 CycleGAN 模型，两套生成器分别负责风景到水墨、人像到工笔的风格迁移，并配合图文模型为视觉结果生成一句诗意注脚。 两个通路均已上线，分别绑定权重 cyclegan-landscape.pt 与 cyclegan-figure.pt。

下载风景权重 下载人物权重
模块 A · 水墨风景

风景 → 水墨

该通路使用 CycleGAN 结构中的 G_A，权重版本 cyclegan-landscape.pt。上传一张风景图，生成器会输出水墨风骨，再通过 G_B 完成循环一致性预览。

服务端路由：POST /api/style-transfer/ink
模块 B · 工笔人像

人物 → 工笔

同样使用 CycleGAN 的第二条支路，目前仍在训练数据集中迭代。界面允许提前接入 API，一旦权重准备好，启用按钮即可开始推理。

服务端路由：POST /api/style-transfer/gongbi
模块 C · 图文生成

图像 → 一句诗

任意选一张图像，调用图文模型抽取意境关键词，并生成一句七言短句，用于在前端同步展示或附加到分享卡片中。

图文路由：POST /api/poem
